## How to Use

1. **Choose your stack** based on your primary role
2. **Read the guide** for setup instructions and tips
3. **Copy the config** from the linked configuration file
4. **Customize** by adding or removing servers

## Structure

Each stack guide includes:

- **Overview** - What the stack is for
- **Recommended Servers** - Categorized with priorities
- **Quick Setup** - Copy-paste configuration
- **Use Cases** - Real-world examples
- **Tips** - Best practices for that workflow
